Critical — read before first use
🛡 Back up your folders first

Before you start working with the program, make a full backup of the following folders from your Ketron device:

• USER_VOICE
• USER_STYLE
• REGISTRATION
• PERFORMANCE

Copy those folders to a safe location on your computer (external drive, separate folder, cloud storage). If anything goes wrong during editing, you will always have the original version to restore from.

It is strongly discouraged to open the USER_VOICE, USER_STYLE, REGISTRATION or PERFORMANCE folders directly from the Ketron Event keyboard or sound module via USB. A mistake or accidental operation can delete an entire folder or its contents.

3
Smart intruder detection
🛡 Smart protection

Event Studio Pro uses smart technology that automatically detects any files which do not belong in the USER_VOICE, USER_STYLE, REGISTRATION or PERFORMANCE folders — for example text files, images or any other file types the Ketron Event device does not recognise. These are called "intruders".

When you load one of these folders into the program, any intruders are shown to you and the program offers three options for each of them:

  • Delete — permanently remove the unauthorised file
  • Move — relocate the file somewhere else on your PC (outside the folder)
  • Ignore — leave it as is (dismiss the warning for this session)

This mechanism prevents unauthorised files from causing potential problems with the Ketron device during playback or loading — for example freezes, errors or unexpected behaviour caused by foreign files that should not be there.

3
Smart detektering av "inkräktare"
🛡 Smart skydd

Event Studio Pro använder smart teknik som automatiskt upptäcker alla filer som inte hör hemma i mapparna USER_VOICE, USER_STYLE, REGISTRATION eller PERFORMANCE — till exempel textfiler, bilder eller andra filtyper som Ketron Event-enheten inte känner igen. Dessa kallas "inkräktare".

När du laddar en av dessa mappar i programmet visas alla inkräktare och programmet erbjuder tre alternativ för var och en:

  • Delete — ta bort den obehöriga filen permanent
  • Move — flytta filen någon annanstans på din PC (utanför mappen)
  • Ignore — lämna den som den är (avvisa varningen för denna session)

Denna mekanism förhindrar att obehöriga filer orsakar eventuella problem med Ketron-enhetens drift under uppspelning eller laddning — till exempel frysningar, fel eller oväntat beteende orsakat av främmande filer som inte borde vara där.

Webbläsar- och antivirusvarningar — läs noga
📥 Webbläsarvarningar vid nedladdning

När du laddar ner EventStudioPro.exe från nedladdningslänken kan din webbläsare (Chrome, Edge, Firefox osv.) visa en eller flera varningar. Dessa är alla normala för all ny programvara och betyder INTE att filen är osäker.

Du kan se något av följande meddelanden:

  • "EventStudioPro.exe isn't commonly downloaded" — webbläsaren varnar om filer som ännu inte har laddats ner många gånger
  • "Confirm you are not a robot" — en extra säkerhetskontroll av webbläsaren
  • "This file may be dangerous" eller liknande — allmän varning för alla .exe-filer

Så här slutför du nedladdningen (beror på din webbläsare):

  • Microsoft Edge: klicka på nedladdningsikonen i verktygsfältet uppe till höger. Om du ser en "See more"-knapp bredvid varningen, klicka på den och välj "Keep" eller "Keep anyway". På äldre Edge-versioner, för musen över varningen tills tre prickar (⋯) visas, klicka på dem och välj "Keep"
  • Google Chrome: klicka på pilen bredvid den blockerade filen i nedladdningsfältet längst ner, välj sedan "Keep"
  • Firefox: klicka på nedladdningsikonen, högerklicka på filen och välj sedan "Allow download"
  • Om du ombeds bekräfta att du inte är en robot, klicka på bekräftelseknappen

🛡 Varför ditt antivirus kan varna dig

När du kör EventStudioPro.exe för första gången kan Windows eller ditt antivirus visa en varning som "Windows protected your PC" eller "Unrecognized app". Detta är helt normalt för all ny programvara som ännu inte har byggt upp en nedladdningshistorik hos Microsoft.

Event Studio Pro är inte ett virus. Varningen visas bara eftersom programmet är nytt och ännu inte har en allmänt erkänd digital signatur. Alla små programvaruprojekt går igenom denna fas under den första utgivningsperioden.
